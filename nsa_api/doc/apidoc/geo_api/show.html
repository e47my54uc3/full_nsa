GET /location
Get location data specific to a User.

Errors

408 Request timeout
404 User location data unavailable

Examples

GET /location?first_name=Rosamond&ip=50.250.223.177&last_name=Tromp
200
{
  "first_name": "Rosamond",
  "last_name": "Tromp",
  "phone_distance_from_ip": 17945.3837,
  "stated_distance_from_ip": 13992.9633,
  "stated_distance_from_phone": 5165.9513
}
